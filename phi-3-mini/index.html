Phi-3-mini導入 (2024/07/16)¶

概要¶

  • Microsoftから2024年4月23日にリリースされた小規模LLM。
    • 通常モデルだと時間はかかるが、GPT-3.5よりも高精度という噂。
      • レスポンスを重視し、普段使いは4bit量子化モデルを想定。

ローカルで大規模言語モデルを利用するためのWebUI導入¶

① 任意の場所からターミナルを開き、下記のコマンドを実行。¶

git clone https://github.com/oobabooga/text-generation-webui.git
cd text-generation-webui
./start_windows.bat
  • Gitクライアントはあらかじめインストールしておく。

② インストール途中でGPU選択が表示されるので、利用する環境を選択。¶

A) NVIDIA
B) AMD (Linux/MacOS only. Requires ROCm SDK 5.6 on Linux)
C) Apple M Series
D) Intel Arc (IPEX)
N) None (I want to run models in CPU mode)
  • 多分上のほうを利用できると高速です。(検証環境は「RTX2060」のため「A」を入力)

③ インストールが正常に完了したら、Webブラウザで「http://127.0.0.1:7860」を開く。¶

  • 他のソフトがポートを占有している場合、画面に表示されているURLを開く。

Phi-3-miniのダウンロードと反映¶

① 「Model」タブを開いて「Download model or LoRA」に「microsoft/Phi-3-mini-4k-instruct-gguf」と入力し「Get file list」ボタンを押下。¶

  • 成功したら「Hugging Face」に登録されているファイルリストが表示されるする。

② 「File name (for GGUF models)」と書かれたテキストボックスに「Phi-3-mini-4k-instruct-q4.gguf」または「Phi-3-mini-4k-instruct-fp16.gguf」と入力し「Download」ボタンを押下する。¶

  • ダウンロードできない場合は手順①で表示されたリストからコピペする。

③ 画面左上の「Model」選択プルダウン右の更新(ぐるぐる矢印)ボタンを押下し、 プルダウンから「Phi-3-mini-4k-instruct-q4.gguf」を選択した状態で「Load」ボタンと「Save settings」ボタンを押下する。¶

  • プルダウンに「Phi-3-mini-4k-instruct-q4.gguf」が表示されない場合、WebUIを一度再起動する。

動作確認¶

「Chat」タブを開き、画面下の「Send a message」と書かれたテキストボックスに質問などを入力し「Generate」ボタンを押下する。¶

  • 出力結果の差異はseed値の設定（シードガチャ）レベルかなと思ったのですが、明らかに通常モデルのほうがおかしな文章を書いているのでちゃんと動いてないのかも・・・

通常モデル（レスポンス：2分ぐらい）¶

  • ChatGPTについて説明してください。
    • ChatGPTは、OpenAIによって開発されたテキスト生成機構です。このモデルは、大量のテキストデータから言語モデルを学習し、人間のように会話を取りながら、様々な問題解決やサポートのための回答を提供できます。ChatGPTはTransformersというディープラーニングアルゴリズムに基づいており、自然言語処理の分野で最先端の研究を行っています。
    • それをより詳しく、具体的に説明してください。
